Text Message (SMS) Alerts

 

Command What It Does Example
Get available balance for your account BAL 1234
Provides available text message commands HELP
Unsubscribe from all text message alerts STOP
Note: Text Message Commands are used to send requests to obtain account information. To send a Text Message Command, text the command, followed by a space and the last four numbers of the card number to 7797807384 (ex. BAL 1234 for a balance inquiry).
